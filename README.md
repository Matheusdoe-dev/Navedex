## Get Started

Clone:

```sh
git clone https://github.com/Matheusdoe-dev/Navedex.git
```

Install dependencies:

```sh
npm install

yarn install
```

Development:

```sh
npm start

yarn start
```

Build:

```sh
npm run build

yarn build
```
